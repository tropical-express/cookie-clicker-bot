# Unofficial Auto Clicker for Cookie Clicker

This is an unofficial browser extension created for educational purposes.

It adds a toggleable auto-clicker to Cookie Clicker.

## Features

- Toggle on/off
- Adjustable click speed
- No data collection
- Minimal permissions

## Disclaimer

This extension is not affiliated with or endorsed by Cookie Clicker or its creator.
It does not modify game files or redistribute any assets.

## Privacy

This extension does not collect, store, or transmit any user data.
